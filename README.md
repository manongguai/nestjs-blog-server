### 六、开发脚本 🧩

- 全局安装turbo后可以直接执行turbo中的脚本

* **快速调试：**

```javascript
 pnpm dev
```

- **打包：**

```javascript
 pnpm build
```


- **test：**

```javascript
 pnpm test
```

- **添加版本变更：**

```javascript
 pnpm changeset:add
```

- **变更版本：**

```javascript
 pnpm changeset:version
```

- **发布包：**

```javascript
 pnpm changeset:publish
```
